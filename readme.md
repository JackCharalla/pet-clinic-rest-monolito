# REST Spring PetClinic (spring-framework-petclinic extension)
Es un proyecto educativo que simula el backend de un aplicativo para Veterinarias.
El proyecto sirve de Apis para la gestión de clientes, veterinarios y mascotas.

## Repositorio GitHUb
[spring-petclinic-rest](https://github.com/spring-petclinic/spring-petclinic-rest)
